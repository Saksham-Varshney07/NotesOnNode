# Node.js Notes App

A simple file-based notes application built using pure Node.js (no frameworks).

## Features
- Add notes
- List all notes
- Read a note
- Delete a note

## Tech Stack
- Node.js core modules (fs, path)

## Usage
```bash
node index.js add "Title" "Note content"
node index.js list
node index.js read <id>
node index.js delete <id>
```
